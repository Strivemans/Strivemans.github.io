时间复杂度与空间复杂度

发表于2021-04-24|更新于2023-04-07
|阅读量:

1.时间复杂度

  • 作用：用于衡量一个算法的优劣，时间次数越短越好

  • 频度：一个算法中的语句执行的次数 称为 频度 记为 T(n)

  • 举例

    由上图可知，一方法的时间复杂度为 n+1 二方法的时间复杂度为 1 由此可知 二方法比一方法的效率高

  • 计算时间复杂度的方法：

    • 忽略常数：比如 2n+10 与 2n的时间复杂度是非常相近的 所以 可以忽略10 直接时间复杂度都等于 2n

    • 忽略低次项：比如 2n²+10n+2 与 2n² 的时间复杂度很相近 所以可以忽略低次项 时间复杂度均为 2n²

    • 忽略系数：当 n² 与 n三次方对比时 可以忽略系数
